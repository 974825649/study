## 标准的文档结构
```html
<!DOCTYPE html>
//文档声明，这是html5
<html lang="cmn-hans">
//根元素， 中文简体
<head>
//文档头
     <meta charset="UTF-8">
     //指定网页内容编码
     <meta name="viewport" content="width=device-width, user-scalable=no, 
     initial-scale=1.0, maximum-scale=1.0, minimum-scale=1.0">
     //适配移动端
     <meta http-equiv="X-UA-Compatible" content="ie=edge">
     //适配IE
     <title>Document</title>
     //显示标题
</head>
<body>
 //网站主体
</body>
</html>
```
